Active Cardiology
Specialists

Retrieves the ID, name, qualification and specialization of all currently active doctors in the Cardiology department. A precise lookup powering the specialist directory.

SQL Query PostgreSQL 
SELECT id, username, qualification, specialization
FROM   doctor
WHERE  specialization = 'Cardiology'
  AND  IsActive = true;
-- Returns: id, name, qualification, specialization
-- Filter:  specialization = Cardiology, IsActive = true
